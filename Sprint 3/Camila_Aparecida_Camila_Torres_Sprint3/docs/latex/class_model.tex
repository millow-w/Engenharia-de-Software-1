\doxysubsection{Detailed Description}
Manages systems and flows, and runs simulations. 

The \doxylink{class_model}{Model} class manages a collection of systems and flows, and can execute a simulation over a specified amount of time. The time progression is controlled by the {\ttfamily run} function.
