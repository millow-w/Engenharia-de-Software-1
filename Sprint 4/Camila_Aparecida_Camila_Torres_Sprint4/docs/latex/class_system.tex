\doxysubsection{Detailed Description}
Represents a system with a specific value. 

The \doxylink{class_system}{System} class holds a value (e.\+g., for simulation purposes) and provides functions to get and set this value. It also includes functions for copying and displaying the system\textquotesingle{}s value.
